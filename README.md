## Installation

<!-- Navigate to the bench directory -->
cd ~/frappe-bench

<!-- Clone the Ant-POS app -->
bench get-app ant_pos https://github.com/anthertech/antPOS.git

<!-- Install Ant-POS on the desired site -->
bench --site yoursite.com install-app ant_pos
